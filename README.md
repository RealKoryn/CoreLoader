# CoreLoader

> Open-source Roblox loading screen with smooth animations, blur effects, and easy customization.

## ✨ Features

- Smooth logo animations
- Dynamic blur effects
- Lightweight & optimized
- Easy to customize

## 📦 Installation

1. Add `LoadingScreen` to `StarterGui`
2. Add `CoreLoader` to `StarterPlayerScripts`

## 🛠 Built With

- Roblox Lua
- TweenService
- Lighting Effects
